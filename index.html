명령어

  1. Git Bash -> 관리자권한으로 실행
  2. cd -> 디렉토리(폴더) 이동
  3. mkdir -> 디렉토리 생성
  4. git init -> 깃저장소로 새로 만듬 (초기화 master)
  5. README.md 파일만들기
  6. git add * / -u / 추가된파일명 -> 작업트리
  7. git commit -m "메세지입력" -> 버전을 만들기위해 스테이지에넣음
  8. git push -> 원격저장소로 보냄
  9. git push origin master
  • $ git clone "https://github.com/K-POPPY/git01"
